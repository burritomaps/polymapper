What is BikeLoca?

If you have a hard time finding out where to park your bike, BikeLoca is for you. It displays a map of all the bike rack locations in Boston.

The data is crowd-sourced by bikers using their iPhone or Android phone. You can submit an existing bike rack or suggest a new one.

The GPS coordinates are transmitted to the CivicAPI server and a photo can optionally be attached. Eventually, we will add support for more metadata, such as the condition of the rack.

How was BikeLoca built?

BikeLoca was built at the Boston Hack Day and was inspired by FixCity, a project by The Open Planning Project, PDXAPI, PDXTrees.org.

We are using data from Open Civic Data (a project by Code for America) and you can see all the API methods at CivicAPI.
